# ImreFS

Simple filesystem using Rust, challenge come from here:
[ImreFS Challenge](https://likeable-launch-ac7.notion.site/ImreFS-Part-1-Interprocess-Communication-b98c36f78771435d8b587b339630ad70)

- [x] Send message
- [x] Stop process
